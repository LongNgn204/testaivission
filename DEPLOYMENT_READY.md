## Quick Deployment

### 1. Install Dependencies
```bash
npm install
```

### 2. Build for Production
```bash
npm run build
```

### 3. Deploy dist/ Folder
Choose one:
- **Vercel** (recommended - zero config)
- **Netlify** (drag & drop)
- **GitHub Pages**
- **Firebase Hosting**

### 4. Set Environment Variables
```

```

- Verify all pages load
